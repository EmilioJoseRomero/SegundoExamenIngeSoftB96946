# CoffeeMachine - Segundo Examen Ingeniería de Software  
Máquina dispensadora de café con backend en **.NET 8** y frontend en **Vue.js**.

---

## Tecnologías utilizadas

| Componente | Tecnología |
|------------|------------|
| Backend | ASP.NET Core Web API (.NET 8) |
| Frontend | Vue.js 3 + Vite |
| Servicios | Swagger, Axios |
| Logs | ILogger |

---
## Backend 
## Opción 1: Instalación y ejecución desde terminal

Ubicación: `ExamTwo/`

### 1. Restaurar dependencias
```bash
dotnet restore
```

### 2. Compilar
```bash
dotnet build
```

### 3. Ejecutar API
```bash
dotnet run
```
## Opción 2: Instalación y ejecución desde la interfaz de visual communitty

### 1. Abrir .sln (seleccionar ExamTwo como proyecto de inicio)

### 2. Ejecutar con F5 o el botón ▶.

Debe abrir Swagger automáticamente.

---

## Frontend - Instalación y ejecución desde terminal

Ubicación: `CoffeeMachineFrontend/`

### 1. Instalar dependencias
```bash
npm install
```

### 2. Levantar servidor de desarrollo
```bash
npm run serve
```

### 3. Acceder en navegador (Local)
```
http://localhost:8080
```
- Debe asegurarse de compilar tanto el Backend como el FrontEnd en conjunto (Se recomienda en ese orden).
